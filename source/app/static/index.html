Idea Map

Real Data

IdeaMap processes and displays tweets according to location.

Live Data

All our data is streamed live from Twitter with a maximum delay of one minute between processing and display. This means that you get the data as soon as it is ready to be viewed.

Intelligent Data

Once tweets are received from twitter, we process them using a machine learner which attempts to classify the tweet as either positive or negative based on the contents of the tweet.



Accessible Data

Once tweets have been processed server side, they are ready to be loaded into our browser based data visualizer which plots the tweets to a map based on the geo-location data from the tweet.

Visible Data

Once tweets are loaded into the visualizer, any tweet on the map can be selected for reading and reviewing.

Trustworthy Data

Because humans are often more trustworthy than a computer when it comes to understanding the speech of other humans, we provide a confidence rating for each tweet to show how confident we are in the classification.



Correctable Data

In the future we hope to allow users to correct the computer when it is wrong and allow the computer to learn from it's mistake by adding the erroneous tweet to the data that the machine learner is trained on.

Cusom Data

Filter results based using our ever growing list of filters and build your own filtering pipeline to show only the data relevant to you.

Opinionated Data

The visualizer could allow you to do brand name analysis based on geographical location, easily identifying geographic clusters of sentiment with a simple glance of an eye.



National Data

The visualizer could also be used to profile a political debate, allowing watchers to near instantaneously gauge citizens opinions on candidates or debate topics.
